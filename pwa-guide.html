üõ†Ô∏è Troubleshooting Common Issues

"Install" Option Not Showing

Possible causes: Browser doesn't support PWAs, app already installed, not using HTTPS

Solutions:

  • Update your browser to latest version
  • Try different browser (Chrome/Edge recommended)
  • Clear browser cache and reload SpotKin
  • Check URL - must be https://dseeker.github.io/spotkin/
Camera Not Working

Common fixes:

  • Check permissions - look for camera icon in address bar
  • Close other apps using camera (Zoom, Skype, etc.)
  • Restart browser and try again
  • Try different browser or device
  • Check camera hardware - test with other apps
Voice Alerts Not Working

Troubleshooting steps:

  • Check microphone permissions in browser settings
  • Test volume levels - increase device volume
  • Try different voice in app settings
  • Check browser compatibility - some have limited TTS support
  • Test with headphones to rule out speaker issues
